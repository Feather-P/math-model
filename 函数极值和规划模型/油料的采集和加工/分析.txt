x6(最终产品质量) = x1+x2+x3+x4+x5

Restrictions:

对于两种油提炼的量存在两个限制：
x1+x2 <= 200 (1)
x3+x4+x5 <= 250 (2)
对于混合后的硬度存在限制：
3<= (x1*8.8 + x2*6.1 + x3*2.0 + x4*4.2 + x5*5.0)/x6*1.0 <= 6
可以将混合后的硬度转化为两个约束：
-(x1*8.8 + x2*6.1 + x3*2.0 + x4*4.2 + x5*5.0) + 3*x6 <= 0 (3)
(x1*8.8 + x2*6.1 + x3*2.0 + x4*4.2 + x5*5.0) - 6*x6 <= 0 (4)
